# 线上铺货 PC 前端交互说明

## 1. 原则与布局

- 最新需求优先级：100% 准确率专题 > 6 月补充版 > 使用指南 > 1.0/2.0。
- 功能和文案沿用 APP；PC 只把列表、商品编辑、候选集改为左 1/4、中 1/2、右 1/4 三栏并列。
- 三栏独立滚动；不增加 APP 没有的说明区、导航、日志或卡片操作。

## 2. 首次进入

- 按门店+账号+免责版本查询是否已接受。
- 未接受时每次进入都弹免责声明；`已阅读并接受` 5 秒后可点，`不接受，退出页面` 返回上一页。
- 接受后页面底部固定显示 `您已接受免责声明，点击查看`；再次打开无需等待。
- 正文由接口按新/老门店返回。

## 3. 左栏列表

### 公共

- 顶部：`查看教程`、`同步新入库商品`。
- 搜索支持商品编码、名称、厂家、条码；待同步/已同步还要搜索外卖商品资料。
- tab：`未匹配 / 待同步 / 已同步`，均显示数量和专题原文提示。

### 未匹配

- 筛选：`优先确认 / 近期入库 / 只看药品 / 已收录upc`。
- `优先确认` 默认关闭；筛选是开关，不进入商品卡片。
- 卡片只展示商品编码、名称、规格、厂家、批准文号。

### 待同步

- 筛选：`有标红 / 无标红 / 近期入库`；有标红和无标红互斥，近期入库可叠加。
- 卡片展示蓝色 `我的商品` 与黄色 `外卖商品`；差异字段标红，外卖商品名称旁显示 `库存未同步`。
- 单条：勾选后参与批量同步，或点击 `同步此商品到外卖平台`。
- 批量：`全部勾选` 再点一次取消全部；`一键同步勾选项到外卖平台` 同步所有勾选项。

### 已同步

- 卡片继续展示蓝黄对照，外卖商品名称旁显示 `库存同步中`。
- 点击商品后在中栏核对资料、右栏查看当前关联和其他候选。

## 4. 中栏商品编辑

- 顶部显示当前商品编码。
- 商品条码为输入框；只有点击 `商品条码` 四个字或 `点击这里查看详细指南` 打开 UPC 抽屉。
- 字段：品牌名称、通用名、商品名称、商品规格、批准文号、生产厂家、处方分类。
- 任意字段修改后，在处方分类下显示 `重置为初始商品资料`；点击恢复初始值并隐藏。
- 主操作：`已补充，重新匹配外卖商品`。
- 条码编辑后的保存位置由后端改为对码专用扩展字段，前端输入交互不变。

## 5. UPC 抽屉

- 展示两张美团后台参考图，支持输入美团 UPC。
- `我知道了` 只关闭；`已补充，重新匹配` 校验后触发手动对码。
- 校验：6-15 位纯数字；错误文案 `请输入合规upc`。
- 无成功候选且 UPC 已收录时展示原安慰弹窗，不恢复旧 `匹配失败`。

## 6. 右栏候选集

- 图片和文字宽度各 1/2；卡片整体点击，不加查看详情或确定按钮。
- 展示图片、商品名称、UPC、批准文号、厂家、处方分类。
- 标签映射：

| 条件 | 标签 |
| --- | --- |
| 当前中间表关联 | 当前关联 |
| 公共库内 + 高置信度 + 非重复 | 高度相似 |
| 公共库内 + 高置信度 + 重复 | 疑似重复 |
| 公共库内 + 中置信度 | 中等相似 |
| 公共库内 + 低置信度 | 略微相似 |

- 候选资料由接口优先返回美团静态库资料，其次 SPU；前端不自行合并。

## 7. 弹窗与状态流转

- 未匹配候选：非重复弹 `确认建联`，重复弹 `发现为重复商品`。
- 主操作：非最后一条 `匹配并同步，下一条`，最后一条 `匹配并同步`；完成后商品进入已同步。
- 待同步单条/批量同步后进入已同步。
- 点击当前候选只弹 `当前关联标品`；点击其他候选弹重新关联确认，主操作 `重新匹配并同步`。
- `解除当前关联` 确认后清除关系，商品回到未匹配。
- `找不到，先看下一条` 只在未匹配且非最后一条展示。

## 8. 教程

- 点击 `查看教程` 开始固定 10 步浮层，入口改为 `退出引导`。
- 点击 `我知道了` 进入下一步；第 10 步后退出。
- PC 对应高亮搜索、优先确认、列表、商品资料、候选集、标红筛选、待同步卡片、批量同步、已同步对照、解除关联。

## 9. 前后端边界

前端只消费结果，不实现：美团门店商品表、动态库、先用后审假上与 line4、AI 拆分商品名、一致性核准、多候选自动择优、对码专用条码落库。

需求正文对未匹配状态的条件与状态机/截图冲突；实现以状态机和截图为准：未匹配为未建立关系，已匹配但未同步进入待同步。
